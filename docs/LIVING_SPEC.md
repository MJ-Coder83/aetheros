# InkosAI — Living Specification

It is a universal visual development environment where humans and AI collaborate to build, orchestrate, and continuously improve any type of digital interface — from web and mobile apps to desktop applications, TUIs, video/podcast production, game development, and future interfaces.

## Domain Canvas — v5 Vision (Universal Visual Development Environment)

The Domain Canvas is the heart of InkosAI. It is a node-based, visual workspace that supports **every type of digital UI** with no gaps.

### Tiered Support

**Tier 1 (Browser-Native)**: React, Next.js, Vue, Angular, Svelte, Electron, Tauri, Figma, Framer, Storybook — full live editing and natural language control.

**Tier 2 (High-Fidelity Emulation)**: Flutter, React Native, .NET MAUI, SwiftUI, Jetpack Compose, WPF, Qt — AI-assisted rendering and interaction simulation.

**Tier 3 (TUI / Terminal)**: Go (Bubble Tea), Rust (Ratatui), Python (Textual) — Visual TUI Layout Editor + AI Co-Pilot for seamless configuration.

**Tier 4 (Plugin Nodes + Embedded Tools)**: Godot, Unity, Blender, DaVinci Resolve, Adobe Suite, VS Code, and any proprietary tool via plugins. Full agentic integration and cross-tool orchestration.

### Key Canvas Features

- Universal Browser Node with live previews and element tagging
- Visual TUI Layout Editor with AI-assisted configuration
- Plugin Node architecture with Agent Bridge for deep tool integration
- Natural language editing across all UI types
- Real-time simulation, Tape overlay, and AetherGit versioning
- Multi-user collaboration and AI Co-Pilot mode
